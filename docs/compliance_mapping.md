# Compliance Mapping

PASSGUARD LABS aligns with recognized security standards.

## Supported Frameworks
- NIST SP 800-63 (Digital Identity)
- ISO/IEC 27001 (Access Control)
- Zero Trust Architecture principles

## Compliance Approach
- Map platform outputs to controls
- Identify gaps instead of claiming full compliance
- Provide actionable remediation insights

Compliance outputs are advisory and not a certification substitute.
